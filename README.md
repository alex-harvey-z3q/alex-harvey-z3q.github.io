[![Build Status](https://img.shields.io/travis/alexharv074/alexharv074.github.io.svg)](https://travis-ci.org/alexharv074/alexharv074.github.io)

To build:

```text
▶ bundle install
▶ bundle exec jekyll build
```

To run the tests:

```text
▶ bundle exec rake
```
